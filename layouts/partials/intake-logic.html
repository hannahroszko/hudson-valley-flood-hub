Prueba de propiedad para herederos

FEMA acepta estos documentos si no tiene la escritura (necesita solo UNO):

  • Declaración de un funcionario público local
  • Recibos de reparaciones mayores (últimos 5 años)
  • Factura de impuestos sobre la propiedad
{{ else }}

Proving Ownership (Heir's Property)

FEMA accepts these alternatives if you aren't on the deed (need only ONE):

  • Public Official’s Statement
  • Major Repair Receipts (last 5 years)
  • Property Tax Bill or Receipt
{{ end }}
